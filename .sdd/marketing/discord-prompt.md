# Prompt: Discord Post for RAG Challenge Chat

## Context

Sasha is a participant in the RAG challenge competition. He wants to post in the competition's Discord channel about Bernstein — an open-source project that was born as a side effect of this challenge.

## Tone & Style

- Casual, genuine, grateful — NOT promotional or corporate
- Russian-speaking community, post in English (the channel language)
- "Hey folks, this challenge produced cool stuff, here's one of them" vibe
- Acknowledge the community and organizers — the challenge inspired this
- Light, conversational, maybe slightly self-deprecating
- Invitation to collaborate, not a sales pitch
- Don't oversell. Don't list every feature. Keep it human.

## Key message points (from Sasha's own words)

1. The challenge produced many artifacts/side-projects — thanks to the community for that
2. Bernstein is one such artifact he wants to share
3. It's open source, contributors welcome — PRs, ideas, whatever
4. Brief mention of what it does (parallel AI agents, one command, tests verify)
5. Not a polished product launch — just a builder sharing something with fellow builders

## What Bernstein is (for context, don't dump all of this)

- CLI tool: `pipx install bernstein`
- One command spawns parallel AI coding agents (Claude Code, Codex CLI, Gemini CLI)
- Deterministic Python orchestration — zero LLM tokens on coordination
- Agents work, janitor verifies (tests pass, files exist), auto-commits
- Used during the RAG challenge to iterate on pipeline experiments overnight
- GitHub: https://github.com/chernistry/bernstein
- Has `good-first-issues` labels, hacktoberfest tags, open to contributors

## What NOT to do

- Don't sound like a Product Hunt launch
- Don't list technical specs or architecture details
- Don't use marketing language ("revolutionary", "game-changing")
- Don't make it long — 4-6 sentences max
- Don't forget the collaborative invitation at the end

## Output

Write 2-3 short variants of a Discord message (English). Each should be 4-6 sentences. The feel should be like a message from a friend in a group chat sharing something cool they made, not an announcement.
